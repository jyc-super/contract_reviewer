🗄️ Supabase (데이터베이스)
계약서, 조항, 분석 결과를 저장하는 데이터베이스 + 파일 저장소
✓ 연결됨
Supabase URL 필수
supabase.com → 프로젝트 설정 → API → Project URL
Service Role Key 필수
프로젝트 설정 → API → service_role 키 (anon 키 아님)
✓ Supabase 연결 성공 — 7개 테이블 확인, pgvector 활성화 확인
📖 Supabase 설정 방법 보기 ▾
  1. supabase.com에서 무료 계정 생성 → New Project
  2. Region: Northeast Asia (Seoul)
  3. Settings → API 탭에서 Project URL과 service_role 키 복사
  4. SQL Editor에서 CLAUDE.md의 스키마 SQL 실행
💡 무료 티어: 500MB DB, 1GB Storage. 개인 프로젝트에 충분합니다.
🤖 Gemini API (AI 분석)
조항 리스크 분석, FIDIC 비교, 구역 분류 등 모든 AI 판단을 수행
✓ 연결됨
Gemini API Key 필수
Google AI Studio → Get API Key → Create API Key · 신용카드 불필요
✓ Gemini 연결 성공 — 무료 티어, 3.1 Flash Lite 500 RPD, Gemma 27B 14,400 RPD
📖 API 키 발급 방법 보기 ▾
  1. aistudio.google.com 접속 (Google 로그인)
  2. Get API Key → Create API Key 클릭
  3. 생성된 키(AIzaSy...)를 위 입력란에 붙여넣기
💡 무료 티어로 하루 약 6건 계약서(40조항 기준) 분석 가능.
