모든 설정, 문서, 에이전트 메모리 파일에서 `D:\coding\contract risk` 같은 절대 경로 사용 금지. 상대 경로(`.venv`, `.claude/agent-memory/...`) 또는 `{프로젝트 루트}` 표기 사용.

**Why:** 절대 경로가 하드코딩되면 다른 컴퓨터/환경에서 프로젝트를 실행할 수 없음. 2026-03-13에 프로젝트 전체에서 절대 경로를 상대 경로로 일괄 수정 완료.

**How to apply:** 새 파일 생성, 문서 작성, 에이전트 메모리 저장 시 절대 경로가 포함되지 않도록 항상 확인. CLAUDE.md 아키텍처 제약 #6으로 명시됨.
